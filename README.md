# learn-html

A beginner-friendly project for learning HTML fundamentals through hands-on examples and small practice pages.

## 1. index.html

The main entry point of the project. This page provides an overview and links to different HTML learning resources and practice files.

## 2. study-calendar.html

A weekly study calendar designed to help learners plan their HTML, CSS, and JavaScript learning schedule with daily topics and activities.

## 3. profile.html

A personal profile page showcasing basic information, skills, and a simple project portfolio. This file focuses on semantic HTML and page structure.

## 4. form-intern.html

An internship application form demonstrating HTML form elements such as input fields, select options, checkboxes, radio buttons, and file upload.

## 5. media-embed.html

A practice page for embedding media content, including images, videos, and audio, using proper HTML tags and attributes.
